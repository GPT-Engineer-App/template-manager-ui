# template-manager-ui

A template manager UI for viewing (jinja, nunchuck) template(s) in source code view, compiled html view, and variables view (source + JSON schema representing template vars.) Templates are stored with meta data for tags, category, name. description. Templates are viewable in a browsable folder tree accessible via the sidebar.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/template-manager-ui.git
cd template-manager-ui
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
